1) Project Overview

Problem: Restaurants need a simple, maintainable system to manage digital menus, including pricing, categories, inventory, and nutrition details.

Our Solution: A modular Restaurant Menu System with three core entities: FoodItem, NutritionStats, and Menu. The system provides ways to add items, display menus, track nutrition, and update stock when orders are placed.
